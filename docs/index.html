Mobile Academic Planning App

Organize terms, courses, assessments, and deadlines in one place.

Academic Planner Pro is a .NET MAUI Android application built for students who need a focused, mobile-first way to manage academic work.

Download APK See Features

Android may prompt you to allow installation from your browser or downloads app before installing the APK.

Core Features

Secure Access

Username and password authentication with optional PIN unlock for faster return access on the same device.

Academic Tracking

Create terms, courses, and assessments with dates, notes, instructor details, and alert settings.

Search & Reporting

Search existing records and view an onscreen report with a title, timestamp, and multiple rows of academic activity.

What the App Includes

  • Term, course, and assessment management
  • Validation for dates, required fields, and instructor email
  • Database-driven status and alert options
  • Search results across terms, courses, and assessments
  • Upcoming academic activity report
  • Local SQLite data storage
  • Optional PIN quick unlock
  • Android APK distribution through GitHub Releases

Installation

  1. Tap Download APK.
  2. Allow the APK file to download to your Android device.
  3. Open the file and allow installation from the current source if Android prompts you.
  4. Install and launch Academic Planner Pro.
  5. Create an account on first launch, then log in to begin using the app.
